- Declaration :

    [const] <type> <name>;

- Assignation :

    <name> << <value>;

- Declaration and Assignation

    [const] <type> <name> << <value>;

- Values

    type  : 
        int -> Integers (no range)
        double -> Non-integers numbers (no range)
        byte -> Integers (-128 <-> 127)
        char -> Characters
        str -> Multiple characters
        bool -> Booleans (true/false)
        auto -> Takes the required type
        udf -> Can have any type

    name  : [a-zA-Z_][a-zA-Z0-9_]*
    
    value : ...

    <> : mandatory
    [] : optional
