- Errors Management

try
  Tries to execute the code in the try block.

catch
  Executes the code in the catch block if an error occurs in the try block.

even
  Executes the code in the even block after the try and catch blocks have finished executing.

raise
  Raises an error of the specifed type.

- Example code :

try {
  raise InvalidNumberLitteral;
} catch {
  // do something
} even {
  // do something else
}
